# Contributing to CollegeFinder

First off, thank you for considering contributing to CollegeFinder! It's people like you that make CollegeFinder such a great tool.

## Where do I go from here?

If you've noticed a bug or have a feature request, please make sure to first check the issues list to see if it has been reported before. If not, open a new issue.

## Fork & create a branch

If this is something you think you can fix, then fork CollegeFinder and create a branch with a descriptive name.

## Implement your fix or feature

At this point, you're ready to make your changes! Feel free to ask for help; everyone is a beginner at first.

## Make a Pull Request

At this point, you should switch back to your master branch and make sure it's up to date with CollegeFinder's master branch:

Then push your branch to your fork and submit a pull request!
